PyTherm
2D Heat Conduction Simulator
v0.5.1  ·  Made by Craig “Duke” Smith  ·  2026
⌖  github.com/dukesmith0/pytherm
PyTherm is an interactive 2D heat conduction simulator for the desktop. Paint a grid of real materials, set fixed-temperature heat sources or constant heat flux sources, and watch Fourier conduction propagate in real time. Built for students, educators, and engineers who want to explore thermal behavior without spinning up a full FEA suite.

Getting Started

1
Requirements
Python 3.10 or newer. PyQt6 and NumPy are the only dependencies.
2
Clone the repository
git clone https://github.com/dukesmith0/pytherm.git
cd pytherm
3
Install dependencies
pip install -r requirements.txt
4
Run
python main.py
A startup dialog lets you configure grid size and ambient temperature, then click Create New Grid to begin.

Features

Real-time FDM solver
Explicit finite-difference with CFL sub-stepping. Harmonic mean conductivity at interfaces.
196 built-in materials
Metals, ceramics, polymers, construction, electronics, gases, liquids, minerals, earth, and more across 26+ subcategories.
Draw, Fill, and Select tools
Paint cells, flood-fill regions, inspect and edit individual or grouped cells.
Heatmap and material views
Toggle between color-by-material and a live temperature heatmap with an auto-scaling legend.
Fixed-temperature sources
Pin any cell to a fixed temperature or set a constant heat flux (W/m²) to model heaters, heat sinks, or boundary conditions.
Steady-state detection
Optionally stop automatically when the simulation converges. Export the view as a PNG.
Cell info tooltip
Hover over any cell for live temperature, energy, thermal time constant (τ), and resistance (R).
Copy / paste cells
Ctrl+C copies a cell's material and temperature; Ctrl+V pastes to all selected cells at once.
Grid coordinates
Row and column indices appear at the grid edges when zoomed in, making large grids easier to navigate.
Paint temperature
Optionally lock the painted temperature when drawing, so new cells start at a chosen temperature.
Four temperature units
Display in °C, K, °F, or Rankine. Type a value with a unit suffix (e.g. 100C, 373K) in any spinbox to auto-convert.
Energy conservation display
Bottom bar shows stored energy E, reference E_ref, and conservation error; confirms solver correctness in real time.
Isotherms and hotspot overlay
Draw temperature contour lines over the heatmap. Highlight cells above a configurable threshold in red.
Protect cells from overpainting
Lock any cell so draw, fill, delete, and paste operations skip it. Toggle with right-click or the P key.
Named cell labels and group highlight
Assign short text labels to cells. Cells with a shared label form a group; selecting one highlights the whole group in orange.
Temperature vs. time plots
Dockable plot panels track selected cells over time. Multiple panels with synchronized cursors and pinned points.
Command palette
Ctrl+Shift+P opens a fuzzy-search palette for all actions and materials -- no menu hunting required.
Grid resize and selection tools
Add or trim rows/cols from any edge. Find the hottest or coldest cell instantly. Reset selected cells to ambient temperature.
